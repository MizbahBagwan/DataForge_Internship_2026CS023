# Project Report – Phase 3 (Excel EDA Dashboard)

## Overview

An Excel dashboard was created using PivotTables, PivotCharts, slicers, and KPI cards to analyze the sales dataset.

## Summary Statistics

The following statistics were calculated for key numeric columns:

- Count
- Sum
- Average
- Minimum
- Maximum
- Median
- Standard Deviation

## Dashboard Components

- Pivot Table 1: Performance by Category and Segment
- Pivot Table 2: Performance by Region, State, and City
- Pivot Table 3: Monthly/Yearly Sales Trend

Charts Created:

- Sales by Category
- Sales by Region
- Monthly Sales Trend

Slicers Used:

- Region
- Category

## Report Questions

### 1. Which pivot table gave the most useful insight?

The "Performance by Category and Segment" pivot table provided the most useful insight because it identified which product categories and customer segments contributed the most to sales and profit.

### 2. What changed when you used slicers?

The slicers allowed the dashboard to be filtered interactively by category and region, making it easy to compare different subsets of the data without changing the PivotTables manually.

### 3. What is one limitation of doing this only in Excel?

Excel dashboards work well for small to medium datasets, but they become slower and harder to maintain with very large datasets. They also provide fewer interactive and automation features compared to tools such as Power BI or Tableau.
