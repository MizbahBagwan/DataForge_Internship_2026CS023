## Report Questions

### Which column had the highest data quality risk?

The Date column had the highest data quality risk because inconsistent date formats can lead to incorrect sorting, filtering, and time-based analysis.

### What cleaning decision did you take and why?

I converted the dataset into an Excel Table, verified there were no duplicate or missing records, and formatted date and numeric columns correctly. These changes improve consistency and ensure accurate analysis.

### Why should raw data and cleaned data be kept separately?

Keeping raw and cleaned data separately preserves the original dataset, allows changes to be tracked, makes the cleaning process reproducible, and provides a backup if errors occur during data preparation.
