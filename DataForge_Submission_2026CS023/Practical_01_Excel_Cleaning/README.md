 # DataForge

## Project Overview

DataForge is a data analytics project that uses a retail sales dataset to practice data organization, cleaning, analysis, and visualization. The project follows a structured folder layout to separate raw data, cleaned data, reports, source code, and project documentation.

## Dataset

The dataset contains retail sales transaction records including customer details, product information, sales values, and order dates.

## Objectives

- Organize project files
- Understand the dataset
- Clean and validate the data
- Perform exploratory data analysis
- Generate business insights
- Create dashboards and reports

## Questions the Dataset Can Answer

1. Which products generate the highest sales?

2. Which states generate the most revenue?

3. Which customer segment is most profitable?

4. Which product categories have the highest profit?

5. How do discounts affect profit?

6. Which shipping mode is used most often?

7. Which regions perform best in terms of sales?

8. Which customers contribute the highest revenue?
