Objective

To load the raw dataset using Pandas, inspect its structure, and generate a schema audit report.

Tasks Performed

Viewed column names and data types.
Created a schema audit table containing column name, data type, missing values, unique values, and a sample value.
Exported the schema audit as schema_audit.csv.
Output Files

schema_audit.csv
Conclusion

The dataset was successfully analyzed using Pandas. The schema audit provides a clear summary of the dataset structure and will be useful for further data cleaning and exploratory data analysis.
